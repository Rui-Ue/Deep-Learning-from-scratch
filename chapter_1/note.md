# 1.3 Python インタプリタ

Python の代入は「箱に入れる」ではなく「オブジェクトにタグ,ラベル,名前をつける」というイメージ．
代入文では，オブジェクトを指す変数(タグ,ラベル,名前)の定義が行われる．

> Python は「動的型付き言語」
